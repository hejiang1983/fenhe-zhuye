# 二年级下册《复习与关联》教学网站

## 项目概览
这是一个用于小学数学教学的交互式网站，包含知识图谱导航和多个课时内容页面。

## 技术栈
- 原生 HTML/CSS/JavaScript
- ECharts (用于知识图谱可视化)
- Font Awesome (图标库)
- Google Fonts (字体)

## 项目结构

```
/workspace/projects/
├── index.html          # 主导航页面(原主程序.html)
├── 知识图.html          # 知识图谱可视化页面
```

## 开发说明

### 本地运行
项目通过 Coze CLI 启动，默认在 5000 端口：
```bash
coze dev
```

### 修改导航链接
如需修改页面跳转逻辑，编辑对应 HTML 文件中的 `onclick="window.location.href='xxx.html'"` 属性。

### 修改样式
- 主页面样式：直接在 index.html 的 `<style>` 标签内修改
- 知识图谱页面：知识图.html 中的 `<style>` 标签
- 其他子页面：每个文件内嵌样式，修改时需单独处理

## 特色功能

1. **知识图谱可视化** (知识图.html)
   - 使用 ECharts 实现可交互的知识结构树
   - 支持鼠标滚轮缩放、拖拽操作
   - 一键展开/收起功能

2. **响应式设计**
   - 主页面采用 CSS Grid 布局，适配不同屏幕尺寸
   - 卡片式设计，清晰的视觉层次

3. **统一的导航体验**
   - 主页面提供清晰的内容分类导航
   - 子页面统一返回入口，避免用户迷失

## 部署说明
项目已配置好 `.coze` 文件，可直接部署到 Coze 平台。
